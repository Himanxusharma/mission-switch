## Adding your own notes

Drop more `.md` files in this folder anytime (e.g. company-specific problem lists). The **clean / refine** scripts walk all six topic folders and skip **`README.md`** (see [STUDY_GUIDE](../docs/STUDY_GUIDE.md)).

## Regeneration

`dsa-patterns-and-practice.md` is one of the files written from **`All_chats.md`** by **`split_all_chats.py`**. Full command order and layout options are in [STUDY_GUIDE](../docs/STUDY_GUIDE.md). After a split, you can optionally run:

`python3 scripts/beautify_playbook_sections.py`

to normalize **At a glance** blocks across the topic folders.
